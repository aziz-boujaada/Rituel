# Premium Animation Guidelines

Goal:

Transform the website into a luxury restaurant experience while keeping animations almost invisible.

General Rules

- Mobile-first
- Elegant
- Calm
- Premium
- Minimal
- Smooth
- Never distracting

Animation Duration

- 300ms–900ms
- Continuous animations only when necessary

Performance

Use only:

- transform
- opacity

Use GPU acceleration.

Avoid multiple simultaneous animations.

Avoid unnecessary Motion components.

Avoid unnecessary re-renders.

Trigger animations once whenever possible.

Respect prefers-reduced-motion.
